# Required Options

```yaml 
base_url: http://localhost
data: json://data.json
(...)
```

## base_url

The base url of the api server.  
Type: `string`

## data

The data file path. Data file is a json file that contains the data used in the test cases.  
The path can be a relative path.  
Type: `string`
